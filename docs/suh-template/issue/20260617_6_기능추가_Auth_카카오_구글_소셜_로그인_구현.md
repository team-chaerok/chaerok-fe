## 📝 현재 문제점

- 현재 앱에 로그인 기능이 없어 사용자 인증이 불가능합니다.
- 카카오, 구글 소셜 로그인 연동이 구현되지 않은 상태입니다.

## 🛠️ 해결 방안 / 제안 기능

- 카카오 로그인 SDK(`kakao_flutter_sdk_user`)를 활용한 카카오 소셜 로그인 구현
- Google Sign-In 패키지(`google_sign_in`)를 활용한 구글 소셜 로그인 구현
- 로그인 화면 UI 구성 (카카오/구글 로그인 버튼)
- 로그인 성공 후 액세스 토큰/리프레시 토큰 저장
- 인증 상태 관리 (로그인/로그아웃 상태)

## ⚙️ 작업 내용

- 카카오 로그인 패키지 추가 및 네이티브 앱 키 설정
- 구글 로그인 패키지 추가 및 SHA 인증서 설정
- 소셜 로그인 화면 UI 구현
- 토큰 로컬 저장 처리 (flutter_secure_storage 등)
- 인증 상태 전역 관리 (상태관리 도구 연동)
